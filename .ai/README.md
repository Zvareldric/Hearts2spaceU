# .ai/ — AI Collaboration Workspace

Folder ini menyimpan aset untuk berkolaborasi dengan AI Pair Programmer (*Technical Lead*). Tujuannya membuat cara kerja dengan AI menjadi **konsisten**, **dapat diulang**, dan **terdokumentasi**.

> Aturan kolaborasinya ada di [`docs/08_ai_guidelines.md`](../docs/08_ai_guidelines.md).

## Isi Folder
| Folder | Fungsi |
|--------|--------|
| [`prompts/`](prompts/) | Kumpulan prompt siap pakai yang terbukti berguna. |
| [`templates/`](templates/) | Kerangka/boilerplate agar output konsisten. |
| [`context/`](context/) | Potongan konteks project untuk "menyuapi" AI. |

> Folder-folder di atas boleh **kosong** di awal. File `README.md` di tiap folder berfungsi menjaga folder tetap terlacak di version control sekaligus menjelaskan fungsinya.
