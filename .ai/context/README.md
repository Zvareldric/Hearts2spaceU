## Contoh isi nanti
- Ringkasan arsitektur (versi pendek).

- Daftar keputusan penting (tautan ke `docs/`).

## Catatan
- Jaga agar isinya **ringkas & terkini** — konteks usang bisa menyesatkan.
- Jangan menaruh **secret** (password, API key) di sini.
